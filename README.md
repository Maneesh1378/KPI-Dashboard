# KPI-Dashboard
This project demonstrates a Power BI-based Sales Financial Dashboard used to analyze company sales performance and financial metrics across different segments, products, and countries.

Project Overview

The dashboard visualizes sales data to monitor key business metrics such as total sales, profit, product performance, and regional sales distribution. It helps stakeholders quickly understand financial performance and identify growth opportunities.

**Key Features**

Total Sales KPI

Total Profit Analysis

Sales by Country Visualization

Product Performance Tracking

Segment-wise Sales Distribution

Interactive Filters and Slicers

**Key Insights**

Certain products generate higher profit compared to others.

Some regions contribute significantly to total sales revenue.

Segment-wise analysis helps identify the most profitable customer segments.

Monthly trends reveal fluctuations in sales performance.

**Tools Used**

Power BI
Data Visualization
DAX Calculations
Data Modeling
Dashboard Design
